# KiWi.MainFrame
Web portal for all IO for all KiWi products

## Stack
- NodeJS
- Express
- React
- Reflux
- MongoDB
- EJS
- Socket.IO
- Scribe.js
- Jest (front-end testing)
- Should.js (back-end testing)
